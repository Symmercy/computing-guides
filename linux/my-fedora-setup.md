# My fedora setup
So, recently I have bought a thinkpad t14 gen 6, it is quite simply and amazing laptop, but, I am not under any circumstances (unless under a VM) using windows on that thing. So the obvious choice is linux then. For a long while I have been an arch user *btw* but nowadays I actually need to get some work done, so that was not a viable possibility. The best middle-ground between stability and the newness I love oh-so much is of course fedora. But not any fedora, I do not want to deal with slow DEs like kde and gnome (which I still adore and use on other machines), so I picked mangowm + noctalia, thats what all the kids do nowadays. And I get them, it is an amazing setup.
